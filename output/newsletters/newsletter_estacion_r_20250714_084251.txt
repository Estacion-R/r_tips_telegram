**📊 Tidy Plots for Scientific Papers: Herramienta para visualización de datos científicos**

Tidyplots es una herramienta diseñada para simplificar la creación de gráficos listos para publicaciones científicas. Con una sintaxis intuitiva, permite ajustar componentes de los gráficos gradualmente. Desde gráficos de barras hasta mapas de calor, podrás crear visualizaciones impactantes para tus investigaciones.
🌐 [¡Explora más en Tidy Plots for Scientific Papers!](https://jbengler.github.io/tidyplots/)

**🛠️ How to open files, folders, websites in R: Optimiza tu flujo de trabajo en R**

Descubre diferentes funciones y herramientas en R para abrir archivos, scripts y proyectos de manera eficiente. Desde utilidades base como `utils::file.edit()` hasta trucos con el paquete `cli`, este recurso te ayudará a acelerar la apertura de tus archivos y mejorar tu productividad en R.
🌐 [Encuentra más detalles en How to open files, folders, websites in R](https://masalmon.eu/2025/06/25/open-files-folders-websites/)

**📚 Interface to the World Database on Protected Areas: Acceso a bases de datos de conservación**

Con wdpar, podrás acceder a datos de la World Database on Protected Areas, utilizada para monitorear áreas de conservación en todo el mundo. Este paquete R también facilita la limpieza de datos según las mejores prácticas, brindándote una interfaz eficiente para trabajar con información crucial para la conservación ambiental.
🌐 [Descubre más en Interface to the World Database on Protected Areas](https://prioritizr.github.io/wdpar/)
